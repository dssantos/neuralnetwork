
# neuralnetwork

## How to dev
```bash
git clone https://github.com/dssantos/neuralnetwork neuralnetwork
cd neuralnetwork
python -m venv .neuralnetwork
source .neuralnetwork/bin/activate
python -m pip install -U pip
pip install -r requirements.txt
```
